# Drip Happens Restaurant Menu

Digital menu hosted on GitHub Pages - scan QR code to view.

## Quick Start

1. Visit the menu: [https://YOUR-USERNAME.github.io/drip-menu](https://YOUR-USERNAME.github.io/drip-menu)
2. Scan QR code from your table
3. View menu fullscreen on your phone

## Setup

1. Create a new GitHub repository called `drip-menu`
2. Push this code
3. Enable GitHub Pages (Settings → Pages → Source: main branch)
4. Done! Your menu is live

## Files

- `index.html` - Fullscreen PDF viewer
- `menu.pdf` - Restaurant menu (6.1 MB)
